Investigacion de HTML5

¿Qué es HTML y que significan las siglas?

HTML es un lenguaje de marcado de hipertexto que interpreta el navegador web para mostrar los sitios o aplicaciones web. Sus siglas significan "HyperText Markup Language" que en español sinifica "lenguaje de marcado de hipertexto".

¿Qué es una pagina web?

Una página web es un documento o información electrónica capaz de contener texto, sonido, vídeo, programas, enlaces, imágenes y muchas otras cosas, adaptada para la llamada World Wide Web (www) y que puede ser accedida mediante un navegador web.

Requerimientos basicos para la elaboración y visualización de una pagina en internet

Lo esencial que se tiene que tener es el conocimiento de como elaborar la pagina web de manera "local" por asi decirlo, en donde podriamos utilizar diversas tecnologias como HTML5, CSS3 y JavaScript para el Front-end y PHP para el Back-end entre otras, ademas es necesario vincularla con bases de datos en caso de que sea necesario.
Una vez listo el codigo se sube a un hosting que es como un almacenamiento en la nuve donde se guardaran los archivos del sitio, teniendo esto es necesario el nombre de algun "Dominio" que basicamente es el nombre (link) con el que el publico en general podra entrar.
Aunque existen vias mas cortas como los "Web Builder" como el famoso WIX que nos facilitan mucho el trabajo.

¿Cuál es la estructura básica de una página web elaborada en HTML?

Cada página comienza con: HTML, A continuación viene la cabecera, delimitada por HEAD y /HEAD, Después, el comando BODY, que indica el comienzo del cuerpo de la página. Las instrucciones HTML se escribirán a continuación, y finalizarán con /BODY . La página acabará con /HTML

¿Cuál es el código para poner una imagen de fondo?

En el documento CSS:
background-image: url(nombre_de_la_imagen);

¿Cuál es el código para colocar una imagen en el cuerpo de la página?

En el documento html:
<img src="nombre_de_la_imagen">

¿Cuál es el código para crear un hipervínculo o enlace?

En el documento html:
<a href="nombre_de_enlace">
Puede ser un archivo local (.html) o una pagina externa (www.paginaweb.com)
